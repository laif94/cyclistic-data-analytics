# cyclistic-data-analytics

Título del Proyecto: Caso de Estudio: Optimización de Estrategias de Marketing para Cyclistic.

El Problema de Negocio: Explicar brevemente el objetivo de Lily Moreno (diseñar estrategias para convertir usuarios casuales en miembros anuales).

Tecnologías Utilizadas: Destacar el stack técnico avanzado que usaste: Python, DuckDB (procesamiento columnar en memoria RAM para millones de filas), Google Colab y Power BI.

Pipeline ETL y Auditoría de Calidad: Mencionar que procesaste un volumen bruto de más de 5.8 millones de filas y que tras una limpieza rigurosa de nulos y control de tiempos (viajes menores a 1 min o mayores a 24 hrs), consolidaste la base en 3,839,852 registros limpios. Esto demuestra tu criterio de gobernanza de datos.

Hallazgos Clave (Insights): Resumir lo que descubriste (los casuales explotan los fines de semana en verano con viajes muy largos; los miembros son de rutina diaria de lunes a viernes).

Capturas del Panel: Puedes guardar tus gráficos de Power BI como imágenes, subirlas al repositorio e incluirlas en el texto para que el proyecto se vea interactivo de inmediato.
